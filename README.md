# BLE Music Pedal

譜めくり用の BLE フットペダル。ペダルを踏んで離すと、BLE HID キーボードとして iPad に右矢印キー(→)を送信する。演奏中や作業中に手を使わず、ペダル操作だけで譜面や資料のページをめくれる。

制作の経緯や解説は Qiita 記事「[BLEフットペダルで譜めくり専用キーボードを自作した話](https://qiita.com/tmurakam99/items/b9e8a31880518a5a0bec)」にまとめている。

## 特徴

- ペダルのリリースエッジで右矢印キーを1回送信(踏みっぱなしでは連続送信しない)
- 誤操作防止の10秒クールダウン
- 30分間無操作で自動スリープ、ペダルを踏むと復帰・自動再接続(ボンディング)
- LiPoバッテリー駆動、USB-C充電、BLE Battery Service経由でiPadに残量報告
- ディスプレイ・ボタンなしでも使えるペダル連打ジェスチャーでのBLEペアリング解除

## ハードウェア

- Seeed Studio XIAO nRF52840
- フットペダル(機械式ON/OFFスイッチ)
- LiPoバッテリー
