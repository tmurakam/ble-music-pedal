## 接続図

### フットペダル（D0 – GND 間）

```
XIAO nRF52840                    フットペダル
                                ┌──────────┐
  D0 ─────────────────────────── NO 端子   │
                                │           │
 GND ─────────────────────────── COM 端子  │
                                └──────────┘
```

| ペダル状態 | D0 電圧 | 動作 |
|---|---|---|
| 離している（開放） | HIGH (3.3V) | 内部プルアップで保持 |
| 踏んでいる（短絡） | LOW (0V) | D0 – GND 間が導通 |

- 抵抗は不要（`code.py` で `Pull.UP` を有効化済み）
- フットペダルの端子に極性なし（どちらの線をどちらに繋いでも動作する）

---

### LiPo バッテリー（基板裏面パッド）

```
LiPo バッテリー          XIAO nRF52840（裏面）
┌──────────┐
│ ＋（赤） ─┼──────────── BAT+
│ −（黒） ─┼──────────── BAT−
└──────────┘
```

- USB-C 接続中は自動充電（外部充電モジュール不要）
- バッテリー駆動中は USB-C を外して使用可能
- **LiPo の逆接は基板破損の原因になるため極性に注意**

---

## 接続まとめ

| XIAO nRF52840 | 接続先 |
|---|---|
| D0 | フットペダル NO 端子 |
| GND | フットペダル COM 端子 |
| BAT+（裏面） | LiPo バッテリー ＋端子（赤） |
| BAT−（裏面） | LiPo バッテリー −端子（黒） |

---

## ペダル動作の電気的説明

```
3.3V
 │
 ├─── 内部プルアップ抵抗
 │
 D0 ───────────── NO端子
                  │ペダル
GND ───────────── COM端子
```

ペダルを踏む → NO-COM 間が導通 → D0 が GND に引き下げられ LOW を検出  
ペダルを離す → 回路開放 → 内部プルアップにより D0 は HIGH を維持
